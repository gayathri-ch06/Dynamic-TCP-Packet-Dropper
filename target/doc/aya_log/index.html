A logging framework for eBPF programs.

This is the user space side of the Aya logging framework. For the eBPF side, see the aya-log-ebpf crate.

aya-log provides the EbpfLogger type, which reads log records created by aya-log-ebpf and logs them using the log crate. Any logger that implements the Log trait can be used with this crate.
